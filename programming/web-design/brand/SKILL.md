# Brand

Brand identity, voice, messaging, asset management, and consistency frameworks.

## When to Use

- Brand voice definition and content tone guidance
- Visual identity standards and style guide development
- Messaging framework creation
- Brand consistency review and audit
- Asset organization, naming, and approval
- Color palette management and typography specs

## Brand Voice Framework

Define for every client project:

```markdown
## Brand Voice

**Personality:** [3 adjectives — e.g., "bold, honest, approachable"]

**Tone by context:**
- Marketing copy: [e.g., "confident, benefit-led"]
- Support/docs: [e.g., "clear, patient, direct"]
- Social media: [e.g., "casual, engaging, human"]

**Language rules:**
- USE: [active voice, short sentences, concrete examples]
- AVOID: [jargon, passive voice, excessive adjectives]

**Brand promise:** [one sentence]
```

## Visual Identity Standards

```markdown
## Visual Identity

**Logo:**
- Primary: [full color on white]
- Reversed: [white on dark background]
- Minimum size: 120px wide / 40px tall
- Clear space: equal to logo height on all sides

**Colors:**
- Primary: #XXXXXX (use for CTAs, headlines, key UI)
- Secondary: #XXXXXX (accents, highlights)
- Neutral dark: #XXXXXX (body text)
- Neutral light: #XXXXXX (backgrounds)
- Semantic success: #22C55E
- Semantic error: #EF4444
- Semantic warning: #F59E0B

**Typography:**
- Display: [Font Name] — headlines, hero sections
- Body: [Font Name] — paragraphs, UI
- Mono: [Font Name] — code, data

**Photography style:** [e.g., "natural light, real people, warm tones"]
**Illustration style:** [e.g., "flat vector, 2-color, geometric"]
```

## Messaging Framework

```markdown
## Messaging

**Positioning statement:**
For [target audience] who [need/pain], [Brand] is the [category]
that [key benefit] because [reason to believe].

**Value propositions (ranked):**
1. [Primary — most important to ICP]
2. [Secondary]
3. [Tertiary]

**Proof points:**
- [Concrete result or stat]
- [Client name / case study]
- [Differentiator vs competitors]

**Tagline options:**
- Option A: [short, punchy]
- Option B: [benefit-focused]
- Option C: [evocative]
```

## Brand Consistency Checklist

Before delivering any branded asset:

- [ ] Logo used in correct variant for background
- [ ] Colors match brand palette exactly (no approximate values)
- [ ] Fonts match brand typography (not system fallbacks)
- [ ] Tone matches brand voice guidelines
- [ ] Clear space rules respected around logo
- [ ] No unauthorized color or font combinations
- [ ] All images match photography/illustration style
