# Premium Utilitarian Minimalism UI Architecture

## Core Directive
"Advanced frontend engineering directive for generating highly refined, ultra-minimalist, document-style web interfaces."

Substance over trend. Interfaces that feel like premium publications rather than typical web applications.

## Typography System
- **Serif fonts** for editorial content (display, headlines)
- **Geometric sans-serifs** for UI elements
- **Monospace** for technical/code elements
- Text IS the design — hierarchy through scale, weight, spacing
- Variable font animations for headline interest

## Color System
- **Warm bone backgrounds** — never pure white (#FFFFFF) or pure black (#000000)
- **Desaturated pastels** — used sparingly for semantic meaning only
- **Monochromatic discipline** — one tint, many shades
- Max saturation: 40% on any accent

## Layout
- **Bento grids** — 1px borders as dividers, generous whitespace
- **Structural clarity** — whitespace IS content
- **Asymmetric proportions** — golden ratio over equal columns
- No hero images spanning 100vw — use contained, intentional imagery

## STRICTLY FORBIDDEN
- Gradients (including subtle ones)
- Neon colors
- 3D glassmorphism
- Generic thin-line icon libraries (Lucide, Feather, standard Heroicons)
- Marketing clichés ("Transform your workflow", "Supercharge your team")
- Oversaturated stock photography
- Shadow-heavy cards
- Rounded pill buttons

## Motion
- Animations are "invisible — present but never distracting"
- Transform-based fade-ins only
- Subtle hover state shifts (color, not scale)
- No scroll-jacking

## Icon Strategy
- Use Phosphor Icons or Radix Icons
- OR custom SVG line work
- Never Lucide, Feather, or Bootstrap Icons in premium contexts

## Reference Aesthetic
- Think: Are.na, Stripe Press, Linear Changelog, Rauno.me
- NOT: typical SaaS dashboard, startup landing page
